The Problem

Less tech proficient users are at a higher risk of entering illegitimate scam websites, resulting in the exploitation of their personal data and financial loss. These users lack the knowledge and tools to be able to distinguish a legitimate website.

Timeline- 3 Weeks,
Group Project, Monash University Bootcamps

View More

User Research and Analysis

The most common feedback received from the user research performed was the lack of resources to report scams as well as the need for an automated and easy-to-use preventative tool that required minimal effort from the user to report a scam

View Less

The Solution

Our solution was to create a plug-in that empowers users to identify non-legitimate websites that aims to collect and misuse personal information. The plugin automatically scans sites, detects irregularities, prevents high risk access and reports it to our database.

User Testing

User Testing Goals

  • Identify Potential pain points while navigating the website to add the extension

User Testing Results
